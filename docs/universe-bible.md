# DARK CASTLE 宇宙圣经 · 第一卷

> 状态：写作与美术的唯一事实来源。正文、人物图、地区页面若与本文件冲突，以本文件为准。

## 核心命题

DARK CASTLE 的第一卷写“被保存的事物是否仍然活着”。四境分别把未来、名字、知识与潮汐保存到失去生命：黑曜王庭封存黎明，鹿冠荒野把记忆压进根系，微光秘库把星空囚禁于玻璃，无潮深渊用骨链固定海。黑城堡不裁决答案，只收藏四境无法独自回答的问题。

## 共同时间

- 暮辉纪 427 年，同一轮暗月升起时，四境异常同时开始。
- 黑曜王庭：第十三钟失声，弥拉寻找奥德林。
- 鹿冠荒野：白树第九次开花，空王座选择新的承担者。
- 微光秘库：第九十九次熄灯，玻璃星空开始反向观察城市。
- 无潮深渊：第七码头迎来黑船，第一道潮痕出现。
- 四境故事可以互相留下物证，但主角在第一卷结束前不得跨境会面。

## 世界硬规则

1. 魔法必须付出可追踪的代价，且代价不能在后文无解释撤销。
2. 时间可以被封存，不能被无成本倒转；记忆可以被转移，不能被凭空恢复。
3. 黑城堡只收藏“尚无答案的问题”，不能作为万能解决方案。
4. 人物死亡后不能以完整肉身复活。回声、遗物、梦与他人记忆可以保留影响。
5. 同一件物品的外观、损坏和去向必须在章节状态表中连续。
6. 具名即角色：正文一旦给予姓名，必须在 `src/lib/characters.ts` 登记并拥有独立肖像。
7. 不需要承担人物弧线的群众只以职务、亲缘或群体称谓出现，不随意新增姓名。

## 人物连续性

人物的年龄、固定外貌、动机和不可逆状态均记录在 `src/lib/characters.ts`。第一卷共十六名具名角色，其中白树、来客与骨冠为非人角色，旧王只允许以无脸剪影出现。

特别约束：

- 弥拉在第三幕前不能确定奥德林在钟内。
- 奥德林进入钟后不能肉身返回。
- 赛勒斯的石翼不能飞。
- 伊芙兰的名字对人类记忆逐步失效，但灰耳始终能通过气味认出她。
- 灰耳在《白树下最后一只记名猎犬》终章死亡，之后只能以路线、气味与花的遗产出现。
- 露珂熄灭的最后一盏灯不能再次点燃。
- 七号胸腔的火在终章前不被其他人物目击。
- 来客滴水但留下干足印；它不能直接说谎。
- 诺娅睡眠超过七分钟，现实必然新增一块梦宫结构。

## 八篇长篇结构

每篇中文正文至少 10,000 个汉字，不计标题、标点、界面文案与人物档案。每篇采用 12—16 章，每章均维护四项状态：地点、时间、在场角色、不可逆变化。超过门槛后以叙事密度为先，不为拉长篇幅重复信息。

1. 《从未见过黎明的钟》：调查失踪—发现可能性税—审判庭围塔—敲响第十三钟。
2. 《长翼审判官的第十三次审判》：圣像苏醒—十二份伪判词—自审—公开证词。
3. 《鹿冠荒野的空王座》：白树选王—名字被夺—灰耳拒忘—折冠与王座远行。
4. 《白树下最后一只记名猎犬》：两声呼唤—气味地图—白树交易—守名传承。
5. 《玻璃占星师的最后一盏灯》：反向星图—倒置天文台—知识伦理—群星离开玻璃。
6. 《会书写名字的五烛花园》：黑玫瑰预写—未生者名单—预言拆分—四境送信。
7. 《无潮海上的来客》：黑船靠岸—骨链月井—牺牲一城的请求—把问题驶向黑城堡。
8. 《章鱼骨王冠的梦》：现实增建—共用梦宫—第九十九房—永远差一寸的门。

## 章节发布门槛

- 汉字数达到 10,000。
- 人物姓名全部可在角色库中解析，且每人拥有独立肖像。
- 相邻章节的地点、时间、伤势、持有物与已知信息无冲突。
- 故事首图与至少三个关键情节发生直接关系，而不是抽象氛围图。
- 结局完成本篇人物弧线，同时只留下一个能连接第一卷总谜题的开放问题。
